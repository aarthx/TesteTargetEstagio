4) Dois veículos (um carro e um caminhão) saem respectivamente de cidades opostas pela mesma rodovia. O carro de Ribeirão Preto em direção a Franca, a uma velocidade constante de 110 km/h e o caminhão de Franca em direção a Ribeirão Preto a uma velocidade constante de 80 km/h. Quando eles se cruzarem na rodovia, qual estará mais próximo a cidade de Ribeirão Preto?
IMPORTANTE:
a) Considerar a distância de 100km entre a cidade de Ribeirão Preto <-> Franca.
b) Considerar 2 pedágios como obstáculo e que o caminhão leva 5 minutos a mais para passar em cada um deles e o carro possui tag de pedágio (Sem Parar)
c) Explique como chegou no resultado.

Resposta:

Pela lógica quando os dois se cruzarem, os dois estarão a mesma distância de Ribeirão Preto.
Quanto a esta distância ela pode ser calculada, utilizando a formúla de movimento da física: S = So + vt
considerando que o carro comece no ponto inicial So = 0 e o caminhão no ponto inicial So = 110km
Aqui existem 3 situações quanto ao pedágio:
1 - O caminhão passa os 2 antes de encontrar com o carro (dando uma vantagem de 10 minutos que são 1/6h para o carro) (110km/h * 1/6h = 18,33km)
2 - O caminhão passar por apenas 1 pedágio antes de encontrar o carro (dando uma vantagem de 5 minutos que são 1/12h para o carro) (110km/h * 1/12h = 9,167km)
3 - O caminhão passar pelos pedágios somente depois de encontrar o carro (sem vantagens para o carro)
O valor do So do carro irá depender da situação
Podemos então descobrir o momento t em que se encontram em cada situação igualando a posição final S e portanto suas equações,
tendo então:
Para a situação 1 -> 18,333 + 110t = 100 - 80t
Para a situação 2 -> 9,167 + 110t = 100 - 80t
Para a situação 3 -> 0 + 110t = 100 - 80t
Resultando no seguinte tempo em cada situação (pós-pedágios se houver):
Para a situação 1 -> t = 0,4298h
Para a situação 2 -> t = 0,4781h
Para a situação 3 -> t = 0,5263h
O que resulta em uma distância para Ribeirão preto em cada situação de:
Para a situação 1 -> S = 18,333 + 110 * 0,4298 = 65,61km
Para a situação 2 -> S = 9,167 + 110 * 0,4781 = 61,75km
Para a situação 3 -> S = 0 + 110 * 0,5263 = 57,89km
Obs.: Dependendo do ponto onde se localiza os pedágios, haveria a possibilidade ainda do carro passar pelo caminhão enquanto ele estivese no pedágio mas não foi considerada
